# ☕ Coffee Shop Landing Page

A modern and responsive **Coffee Shop Landing Page** built using **HTML, CSS, and JavaScript**. This project showcases a stylish homepage for a coffee café with featured categories, popular drinks, latest blogs, and a professional footer.
